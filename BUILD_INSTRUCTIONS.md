# Smart Office OS — Build Guide

## Quick Start (Local)
```bash
npm install
npm run build
npm run electron:dev
```

## Build EXE
```bash
npm run electron:build:win
```
EXE milega: `release/Smart Office OS Setup 1.0.0.exe`
